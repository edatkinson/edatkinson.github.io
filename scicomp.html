Scientific Computing

This project involved developing a suite of modules for solving various types of differential equations and conducting advanced mathematical analyses:

  • Ordinary Differential Equations (ODEs): Developed solvers for ODEs applicable to a range of scientific problems.
  • Partial Differential Equations (PDEs): Created modules to solve complex PDEs, useful in fields like physics and engineering.
  • Limit Cycle Analysis: Conducted analysis for systems with periodic solutions, important in nonlinear dynamics.
  • Parameter Continuation: Implemented methods for exploring the behavior of solutions as parameters vary, aiding in the study of bifurcations and stability.

This project provided powerful computational tools for scientific research, contributing to the understanding and modeling of complex systems. It further improved my software development skills and deepened my knowledge of numerical methods and mathematical modeling.

My final report is a jupyter notebook which includes demonstrations on how to apply my modules to relevant problems, an overview of my key software decisions and a self reflection report.
