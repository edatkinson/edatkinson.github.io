Detecting Prohibited Items in Airport X-rays

This project involved the development of an advanced image processing AI to detect prohibited items in Airport X-ray machines. The AI system leverages sophisticated machine learning techniques, including:

  • Multi-Class and Binary Convolutional Neural Networks (CNNs): Used for high-accuracy image classification.
  • Data Augmentation: Enhanced the training dataset with various transformations to improve model generalization.
  • Customized Loss Functions: Tailored the loss functions to better address the unique challenges of detecting prohibited items.
  • Stratified Sampling: Ensured balanced representation of all classes during training.
  • Performance Analysis: Employed rigorous evaluation metrics to optimize model performance.

The outcome was a highly effective image processing AI that significantly improved the accuracy and reliability of prohibited item detection in X-ray images, contributing to enhanced airport security operations.
