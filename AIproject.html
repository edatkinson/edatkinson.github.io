Detecting Prohibited Items in Airport X-rays

This group project focused on developing two Convolutional Neural Network (CNN) classifiers to evaluate the safety of bags passing through X-rays during airport security screening. The primary objectives included building a binary classifier that determines whether a bag is 'safe' or 'not safe' and a multi-class classifier that categorizes unsafe items into specific types, such as 'gun,' 'hammer,' 'knife,' or 'scissors.'

  • Binary and Multi-Class Classification: Developed a binary classifier that achieved near-perfect accuracy (99.7%) in distinguishing safe from unsafe items. The multi-class classifier, however, struggled with categorizing specific prohibited items.
  • Data Handling: The data was preprocessed to address class imbalance using stratified sampling and a custom weighted loss function. All images were resized and normalized for uniform input.
  • Custom Loss Function: Implemented a custom loss function to penalize false-negative predictions more heavily, which is crucial for minimizing security risks.
  • Model Architecture: Both models featured three convolutional layers with batch normalization, max pooling, and fully connected layers, optimized using SGD, Adam, and RMSprop optimizers.
  • Performance Measures: Evaluated model performance using precision, recall, and the F2-score, emphasizing the importance of minimizing type-II errors (false negatives).

The project demonstrated that while the binary classifier was highly reliable, the multi-class classifier required further improvements to accurately categorize prohibited items. Future work could explore expanded hyperparameter searches, data augmentation techniques, and the development of a more robust and generalizable model.
